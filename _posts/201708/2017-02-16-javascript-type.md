Javascript 类型 <!--more-->

+ 五个基本类型（primitive data type) 
    - number，特殊的是NaN，Infinity，-Infinity。 NaN（Not a Number)实际上还是Number类型 
    - string 
    - boolean 
    - undefined，唯一值是undefined 
    - null，唯一值是null  
+ 其他都是object data type. 包括数组
+ 需要注意的一点是：所以值都可转化为true（通过!!两次取非）除了以下六个false值 
    - ""
    - 0 
    - null 
    - undefined 
    - NaN 
    - false
